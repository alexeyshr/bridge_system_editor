# PRIORITIES.md — Приоритизация идей BridgeHub

> Контекст: техническую часть делает **один человек + AI**. MVP должен быть компактным и запускаемым.
> Не строим то, что уже существует — интегрируемся.

## Слои

- 🔴 **Ядро (MVP)** — без этого нет смысла запускать. Цель: рабочий портал, первые 100 пользователей. Срок: 2-3 месяца
- 🟡 **Ключевые (v0.2-0.3)** — делает портал уникальным и полезным. Срок: +2-3 месяца после MVP
- 🟢 **Расширения (v0.4-0.5)** — рост и удержание. Срок: +3-6 месяцев
- 🔵 **Будущее (v1+)** — когда есть аудитория 500+. Срок: 6-12+ месяцев

---

## 🔴 ЯДРО (MVP)

> Минимальный жизнеспособный портал. Контент + профили + сообщество. Запуск для бриджевого комьюнити Москвы.

### Пользователи и профили
- Регистрация (email + Telegram Login Widget)
- Профиль игрока: ФИО, город, bio, аватар, соцсети
- Привязка bridgesport_id (ручная, верификация модератором)
- **Импорт истории с bridgesport.ru** — автоподтягивание турниров, рейтинга, партнёров при привязке. Вау-эффект при регистрации
- Роли: Игрок + Модератор
- Привилегии: пока только Автор (остальные — позже)

### Контент
- Посты (Markdown): статьи, разборы, туториалы, новости
- Quick Takes (короткие заметки)
- Комментарии (древовидные)
- Теги-концепции (базовый набор 30-50: конвенции, техника, форматы)
- Голосование ⬆/⬇ + карма (две оси: контент + репутация)

### Турниры
- Календарь (парсинг bridgesport.ru)
- Результаты (парсинг bridgesport.ru + bridgemoscow.ru)
- Рейтинги ФСБР (парсинг)
- Страница турнира с базовой информацией

### Базовый дизайн
- Светлая тема, минималистичный (LessWrong-стиль)
- Адаптивность: desktop + mobile
- Главная: лента постов (свежее / лучшее)

### Поиск
- Базовый полнотекстовый поиск (посты, игроки)

### Модерация
- Правила портала (code of conduct)
- Кнопка «Пожаловаться»
- Бан/мут для модераторов

### Telegram
- Авторизация через Telegram Login Widget
- Базовый бот: уведомления (ответы на комменты, новые посты по подпискам)

### Техническое
- Next.js + PostgreSQL + Tailwind (один стек, один деплой)
- SEO из коробки (Next.js SSR)
- Docker Compose → VPS

## 🟡 КЛЮЧЕВЫЕ (v0.2-0.3)

> Делает портал уникальным. Привлекает первых 500 пользователей.

### Контент (расширение)
- Серии (Sequences) — структурированные курсы из постов
- Markdown+ расширения: блоки раздач (:::deal), торговля (:::bidding), квизы (:::quiz)
- Визуализация раздач (SVG-компонент)
- **Bidding Polls** 🔥 — еженедельные голосования по торговле
- Библиотека / Concepts Portal (вики-страницы тегов)

### Профили (расширение)
- **Профиль пары** — совместная статистика, стаж, КК
- Партнёрства (текущие/бывшие)
- Графики рейтинга ФСБР по месяцам
- Привилегии: Тренер, Судья, Организатор (с верификацией)

### Задачник
- Интерактивные задачи: торговля, розыгрыш, защита, стартовый выход
- Адаптивный рейтинг (Elo-подобный)
- Задача дня
- **Social proof**: «78% игроков выбрали тот же ход»

### Турниры (расширение)
- **Пост-турнирные отзывы** — организация, судейство, площадка, поведение игроков
- Площадки (Venues) как сущность
- Карта турниров (визуальная карта России)
- Подписка на турнир («Мне интересно»)

### Клубы
- Страницы клубов: описание, участники, расписание
- Карта клубов России («Найди клуб рядом»)
- Клубная лента

### Поиск партнёра
- Доска: «Ищу партнёра на [турнир]»
- Статус пары: «ищу» / «есть» / «открыт»
- Telegram-бот рассылает подходящим игрокам

### Telegram (расширение)
- Бот: действия (голосование, запись на турнир, задача дня)
- Канал: автопубликация лучших постов
- Инлайн-режим: `@BridgeHubBot Шрамков` → карточка

### Karma 2.0
- История кармы (прозрачный лог)
- Бейджи по уровням
- Защита от злоупотреблений (модерация негативных отзывов)

### Прочее
- Персональный календарь игрока + экспорт iCal
- Верификация профиля (галочка)
- Этика и Fair Play — кодекс, подписание при регистрации
- FAQ / База знаний
- Онбординг-тур для новичков

## 🟢 РАСШИРЕНИЯ (v0.4-0.5)

> Рост аудитории, удержание, обучение. 500 → 2000 пользователей.

### AI-функции
- **AI-тренер** 🔥 — загрузи раздачу → получи разбор (DD + LLM)
- AI-авторазбор турнира
- AI-модерация комментариев (токсичность, спам)

### Обучение
- Тренер ↔ Ученик (связь, прогресс)
- Бесплатные курсы (путь новичка: «от 0 до первого турнира за 30 дней»)
- Интерактивные курсы с мгновенной обратной связью (формат GoMagic)
- **«Попробуй бридж за 10 минут»** — интерактивный мини-урок на главной

### Песочница
- Тренировка торговли с AI-ботом
- Тренировка стартовых выходов
- Strategy Sandbox (unrated)

### Задачник (расширение)
- **Bid Rush / Defense Rush** — задачи на время, соревнование
- Лиги задачника (автоматические, по уровням)
- Челленджи и стрики

### Геймификация
- Ачивки / достижения
- Коллекционные бейджи
- Система приглашений (реферал): карма за привод друга

### Турниры (расширение)
- Публикация результатов напрямую (импорт скоринг-файлов, API)
- Live-результаты
- Протоколы и апелляции
- Статистика для организаторов

### Конвенционные карты
- Онлайн-редактор (стандарт WBF)
- Совместное редактирование с партнёром
- Версионирование
- QR-код → онлайн-версия

### Контент (расширение)
- **Deal Studies** — совместный разбор раздач (формат Lichess Studies)
- Bidding Explorer — статистика торговли по базе раздач
- Сравнение систем торговли (2/1 GF vs Precision)
- Библиотека книг (рецензии, рейтинги)
- Master Solvers Club

### Сообщество
- Региональные хабы
- Менторская программа (бадди-система)
- Ежегодный «Рейтинг клубов»
- Опросы и голосования
- Еженедельный дайджест (Telegram/email)

### Telegram Mini App
- Профиль, задачник, маркетплейс — внутри Telegram
- «Попробуй бридж» — прямо в Telegram

### Прочее
- Рекомендательная система («Тебе может быть интересно»)
- Фото и медиа с турниров
- Социальный граф (кто с кем играл)
- Печатные материалы (PDF: КК, раздачи, дипломы)
- Офлайн-режим (PWA)
- Доступность (крупный шрифт, accessibility)

## 🔵 БУДУЩЕЕ (v1+)

> Когда есть аудитория 2000+. Масштабирование, монетизация, влияние.

### Финансовая экосистема
- Внутренний кошелёк (пополнение, вывод, P2P)
- Фонд развития бриджа + прозрачный дашборд расходов
- Оплата: турнирные взносы, курсы, маркетплейс
- Эскроу для платных партнёров
- Юрлицо (АНО), 54-ФЗ, ЮKassa/Тинькофф

### Маркетплейс платных партнёров
- Профессионалы: «Играю за N руб/турнир»
- Фильтрация, рейтинг, отзывы
- Парный matching на основе совместимости

### Платные курсы
- Тренер создаёт курс с ценой
- Виртуальный стол для занятий (Classroom)
- Вебинары / Онлайн-занятия

### Соревновательная вертикаль
- Рейтинговые онлайн-лиги (новички / средние / топ)
- Ladder / Лестница вызовов
- Командные соревнования между клубами
- Национальный рейтинг на портале
- Сезоны и сезонные рейтинги
- Ежегодная «Премия BridgeHub»

### Инструменты организатора
- Создание турниров на портале
- Онлайн-регистрация
- Генератор раздач, таблицы движения, скоринг
- Интеграция с BridgeMate
- Экосистема для начинающих организаторов

### Командный бридж
- Профиль команды, командный рейтинг
- Командные турниры

### Массовые мероприятия
- Всероссийский день бриджа
- Кубок регионов
- Массовый онлайн-турнир
- Выездные мастер-классы в регионах

### Продвижение и медиа
- YouTube-канал, подкаст «Бридж за чаем»
- Стримы турниров, Broadcasts (live-трансляции)
- «Бридж-беседа мастеров» по пятницам
- Статьи на Хабре / VC
- Celebrity Bridge — медийные турниры
- «Матч века» AI vs Эксперт

### Коллаборации
- Шахматы, го, ЧГК — совместные турниры и кросс-промо
- Интеллектуальные фестивали

### Школы, вузы, бизнес
- Программа «Бридж в школе»
- Студенческая лига, бридж-клубы при вузах
- Корпоративные турниры, корпоративная лига
- Стипендии из фонда

### ФСБР и власти
- Раздел «Федерация» на портале
- Привилегия «Функционер ФСБР»
- Помощь с аккредитацией региональных отделений
- Гранты (Президентские, региональные)
- Программа «Активное долголетие»
- Спортивные разряды (КМС, МС)

### Категории игроков
- Юниорский бридж (до 25)
- Сеньорский бридж (60+)
- Женский бридж

### Магазин

### Античит
- Детекция подозрительных паттернов
